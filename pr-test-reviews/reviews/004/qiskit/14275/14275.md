## PR Summary

This pull request (#14275) simplifies the `InstructionDurations` class in Qiskit by removing an unexecutable code path related to the now-removed BackendV1. The `.from_duration()` method has been updated to handle only the BackendV2 case, thereby cleaning up the codebase. The method signature remains unchanged to ensure backward compatibility, preparing for the potential addition of BackendV3 in the future. This change enhances code maintenance and clarity, with test coverage slightly improved to 87.86%.
